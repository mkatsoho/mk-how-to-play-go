# 简介

## 名
- 碁(音其），手淡
- 碁，(音ご (go, Jōyō)); ぎ (gi),
参考：[五十音图](https://zh.wikibooks.org/wiki/%E6%97%A5%E8%AF%AD/%E5%81%87%E5%90%8D)

## 圈地
- 三种评价圈地大小的方法，中/日韩/应氏
- 避免平局

## 手段
- 黑先，轮流落子
- 围地 -> 案例1
- 围死对方的子，提子 -> 案例2

## 案例
- 一半一半
- 我的一半里还有对方的子
